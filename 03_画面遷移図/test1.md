```@
@startuml
[*] --> トップページ
トップページ : do/ジャンルを表示
トップページ -> ジャンル : ジャンルボタンをクリック
トップページ -> リンク : リンクをクリック
ジャンル : do/写真を表示
ジャンル--> トップページ : トップページボタンをクリック
ジャンル -> ジャンルトップページ : 写真をクリック
ジャンルトップページ :do/一覧を表示
ジャンルトップページ --> トップページ : トップページボタンをクリック
ジャンルトップページ -> ジャンル : ジャンルボタンをクリック
ジャンルトップページ --> 詳細 : をクリック
詳細 : do/内容を表示
詳細 --> トップページ : トップページボタンをクリック
詳細 --> ジャンル : ジャンルボタンをクリック
詳細 -> リンク : リンクをクリック
リンク : do/リンク先のサイトの表示
@enduml
```
