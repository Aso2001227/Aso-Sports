```@
@startuml
[*] --> トップページ
トップページ : do/ジャンルを表示
トップページ --> ジャンル : ジャンルボタンをクリック
ジャンル : do/写真を表示
ジャンル--> トップページ : トップページボタンをクリック
ジャンル -> ジャンルトップページ : 写真をクリック
ジャンルトップページ :do/一覧を表示
ジャンルトップページ --> トップページ : トップページボタンをクリック
ジャンルトップページ -> ジャンル : ジャンルボタンをクリック
ジャンルトップページ --> 観光詳細 : 観光名をクリック
観光詳細 : do/案内説明を表示
観光詳細 --> トップページ : トップページボタンをクリック
観光詳細 --> ジャンル : ジャンルボタンをクリック
観光詳細 -> リンク : リンクをクリック
リンク : do/リンク先のサイトの表示
@enduml
``` 
